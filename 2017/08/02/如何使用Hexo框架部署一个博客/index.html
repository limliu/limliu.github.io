如何使用Hexo框架部署一个博客

发表于 2017-08-02

什么是 Hexo

Hexo 是一个快速、简洁且高效的博客框架。并且有很多靓丽酷炫的主题可以选。
使用它，你可以快速的搭建自己的博客网站。

软件安装

使用前需要安装以下软件：

  • Node.js
  • Git

本安装教程只针对windows用户，Mac 用户或 Linux 用户可自行百度。对于windows用户来说，建议使用安装程序进行安装。安装时，请勾选Add to PATH选项。

安装完成之后，接下来只需要在 cmd 里使用 npm 即可完成 Hexo 的安装。npm 是 Node.js 下的包管理器，类似于 Python的 pip。
